<h1> Brain games </h1>
<h3>Brain games is a set of five mini-games. Rules are simple: choose a game, answer correctly 3 times in a row and you win, congratulations!</h3>

<p>Games:<p>
<ol>
    <li>brain-even</li>
    <li>brain-calc</li>
    <li>brain-gcd</li>
    <li>brain-progression</li>
    <li>brain-prime</li>
<ol>
</div>

<div>
<h1>Installation:</h1>
<p>sudo npm link</p>
